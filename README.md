# vue-gantt-linetable

A Vue 3 component for displaying Gantt line tables.

## Installation

```bash
npm install vue-gantt-linetable
```

## Usage

```vue
<template>
  <GanttLineTable />
</template>

<script setup>
import { GanttLineTable } from 'vue-gantt-linetable'
</script>
```

## Development

```bash
npm install
npm run dev
npm run build
```
